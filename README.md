Coding Bootcamp Website 
 A complete 5-page responsive website for a coding bootcamp offering Web Development, Python Programming, and Full Stack Development courses. 
 
Project Structure

coding-bootcamp-project/ 
├── index.html 
├── cssFile/ 
│   └── globalStyles.css 
├── jsFile/ 
│   └── jsFunctions.js 
├── pages/ 
│   ├── about-us.html 
│   ├── course-details.html 
│   ├── dashboard.html 
│   └── registration.html 
└── sources/ 
└── images/ 
 
Pages Included 

Home (index.html)- Hero section, featured courses, dashboard preview 
Courses (course-details.html)- Detailed information about 3 courses 
About Us (about-us.html)-Mission, story, values, contact form 
Dashboard (dashboard.html) - Student progress tracking 
Get Started (registration.html)- Sign in / Sign up forms 
 
Features

- Fully responsive design (mobile & desktop) 
- Flexbox only (no Grid) 
- External CSS only (no inline styles) 
- Interactive JavaScript (menu toggle, form validation) 
- Student progress dashboard with graphical view
